                syntax reference
                
tokens (whitespace-separated)
  PHONEME              ARPABET code (AY IH AA S ...)
  PHONEME!             stressed (+8 Hz lift, longer duration)
  PHONEME+N            rising glide N Hz (pitch sticks)
  PHONEME-N            falling glide N Hz (pitch sticks)
  PHONEME(+N)          transient ornament, pitch reverts after
  ( PHONEMES )         syllable group: share one rate slot
  ,  ;  .              pause 100 / 200 / 300 ms
  bN  b=N  bC4         set base F0 in Hz or note name
  b+N  b-N             relative F0 shift
  b                    reset base F0
  rN                   set rate (ms/phoneme)
  r                    reset rate
  pN                   insert exact N-ms pause
  vN  wN               vibrato depth Hz / rate Hz
  hN                   breathiness 0–1
  tN                   spectral tilt -0.9 (dark) .. +0.9 (bright)
  gN                   vocal effort 0–1 (0.5 default)
  # rest of line       comment
